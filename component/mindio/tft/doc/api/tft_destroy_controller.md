# tft\_destroy\_controller<a name="ZH-CN_TOPIC_0000002543201093"></a>

## 接口功能<a name="zh-cn_topic_0000002012581709_section23222518147"></a>

在训练完成后，调用该接口以关闭MindIO TFT  Controller服务。

## 接口格式<a name="zh-cn_topic_0000002012581709_section1585114371516"></a>

```
mindio_ttp.framework_ttp.tft_destroy_controller()
```

## 接口参数<a name="zh-cn_topic_0000002012581709_section522915911518"></a>

无

## 返回值<a name="zh-cn_topic_0000002012581709_section8785165291317"></a>

无返回值。出错时会打印ERROR日志并抛出异常。
