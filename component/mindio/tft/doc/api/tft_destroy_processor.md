# tft\_destroy\_processor<a name="ZH-CN_TOPIC_0000002543201099"></a>

## 接口功能<a name="zh-cn_topic_0000001976021310_section168711210120"></a>

在训练完成后，调用该接口以关闭MindIO TFT  Processor服务。

## 接口格式<a name="zh-cn_topic_0000001976021310_section7293175278"></a>

```
mindio_ttp.framework_ttp.tft_destroy_processor()
```

## 接口参数<a name="zh-cn_topic_0000001976021310_section135901320671"></a>

无

## 返回值<a name="zh-cn_topic_0000001976021310_section1777516402588"></a>

无返回值。出错时会打印ERROR日志并抛出异常。
